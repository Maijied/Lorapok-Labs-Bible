# Lorapok Labs Bible

> **Building the Future. One Line at a Time.**

The official website and brand guide for the **Lorapok Labs** ecosystem — an independent, open-source ecosystem dedicated to building "Products That Feel Alive."

## Live Site

Deployed to GitHub Pages: [https://maijied.github.io/Lorapok-Labs-Bible/](https://maijied.github.io/Lorapok-Labs-Bible/)

## Features

- **Progressive Web App (PWA)** — Installable, offline-ready, desktop-app UX
- **Biological UI Design System** — Neon-on-dark glassmorphism, cyber-industrial aesthetic
- **Desktop App Feel** — Custom title bar, collapsible sidebar, status bar, command palette
- **7 Pages** — Home, Products, Atlas Deep-Dive, About, Brand Identity, Research, Connect
- **Animated SVG Mascot** — Cybernetic Black Soldier Fly Larva with glowing eyes
- **Responsive** — Full mobile/tablet/desktop adaptive layout
- **Zero Backend** — Fully static, JSON-driven content

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React 18 + TypeScript |
| Build | Vite 8 |
| Routing | React Router v7 (HashRouter) |
| Animations | Framer Motion |
| Icons | Lucide React |
| Styling | CSS Modules + Custom Properties |
| PWA | vite-plugin-pwa |
| Deploy | GitHub Pages + GitHub Actions |

## Products Showcased

1. **Lorapok Atlas** — API Directory (2100+ APIs, 34 categories)
2. **Lorapok Media Player** — Cross-platform audio/video player
3. **Lorapok Laravel Monitor** — Zero-config execution monitor
4. **Lorapok Keyboard** — AI-powered Bangla NLP keyboard
5. **Spotlight Tickets** — Self-hosted event ticketing
6. **Linpad** — Minimalist text editor for Ubuntu
7. **Roast as a Service** — Multi-platform humor API
8. **Lorapok AI Agent** — AI-powered coding assistant

## Development

```bash
cd app
npm install
npm run dev
```

## Build

```bash
cd app
npm run build
```

Output is in `app/dist/`.

## Deployment

Pushes to `main` automatically deploy via GitHub Actions to GitHub Pages.
